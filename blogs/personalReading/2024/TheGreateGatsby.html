    # 1.故事梗概

    《了不起的盖茨比》故事线非常简单，六个关键字就可以概括：办宴会->重逢->摊牌->车祸->枪杀->葬礼。

    穷小子出身的盖茨比，因为一次偶然的机会，认识了富家女黛西，两人很快坠入爱河。但是限于阶层的差异，没能在一起，黛西嫁入同样豪门的汤姆-布坎南。婚后的黛西并不幸福，汤姆在芝加哥拈花惹草，到了纽约之后还在包养情妇。盖茨比选择去了战场，回来之后努力发家致富，成为当时纽约的一大富豪。为了与黛西再续前缘，盖茨比在自己的豪宅里疯狂的办宴会，只为了能吸引黛西前来。

    后来无意间盖茨比得知自己的邻居“我”和汤姆黛西一家有远亲，因此借助“我”去搭线，与黛西重逢相遇。并带着黛西去了自己的家中。

    盖茨比进入到汤姆一家的生活之后，经过几次接触，最终三人摊牌，盖茨比表示黛西会和他离婚再和自己结婚，汤姆则当场揭穿了盖茨比的财富来路不正，他本人其实是满口谎言的卑劣罪犯。

    黛西情绪失控，驾车返回的途中撞死了汤姆的情妇，死者的老公受汤姆的误导认定凶手是盖茨比，一枪杀死了盖茨比。黛西和汤姆却远走高飞，连盖茨比的葬礼都没有参加。最后只有我和盖茨比的父亲出席了葬礼。

    # 2.盖茨比为什么了不起

    # (1) 对黛西的一往情深

    书中的盖茨比是一个非常痴情的青年，在自己发家致富获取了巨额财富之后，依然恋恋不忘自己的白月光，做了很多疯狂的举动。

    a.花费巨资在黛西家的对面买了豪宅，只为了能看到黛西家的绿光(这里也是盖茨比首次出场的画面)。

    b.在自己家里疯狂的办免费的宴会，营造浩大的声势，希望有一天黛西能前来参加哪怕一次，这样二人就会有机会再见面。

    c.黛西在芝加哥居住了一段时间，盖茨比就订阅了当地的报刊，并专门做了一个剪报，收集黛西的各种信息。

    d.与黛西重逢时的精心布置，满屋子的花，以及见面时提到的到十一月就五年整了，可见这五年期间盖茨比一直日期精确地计算着他和黛西分开的日子。

    e.黛西开车撞死了人，他毫不犹豫地要为她顶罪，为她隐瞒真相；

    f.在黛西回家之后，一直担心汤姆知道是黛西开的车撞死了自己的情妇梅朵，会粗暴的对待她，所以一直守在黛西家的外面。

    # (2) 出身底层却不甘与低头，努力破圈

    书中提到盖茨比出身在西部贫苦的人家，家庭几乎不能给他提供任何帮助。后来的盖茨比发迹，靠的是自己聪明的头脑，以及不屈不挠的寻求自己的“美国梦”，去努力跨越阶级。在盖茨比的葬礼上，父亲拿出了盖茨很小的时候的一个每天日程安排：

    起床……………………………………6∶00

哑铃锻炼和爬墙练习…………………6∶15-6∶30

研究电学等等…………………………7∶15-8∶15

工作……………………………………8∶30-16∶30

棒球和其他运动………………………16∶30-17∶00

练习演讲和社交礼仪…………………17∶00-18∶00

研究有用的新发明……………………19∶00-21∶00

总体目标绝不浪费时间去沙福特家或者【某个姓，字迹看不清楚】绝不吸卷烟或者嚼烟叶

    可以看到从小盖茨比就是一个非常自律，有远大目标理想的孩子，出身贫寒，奋发向上，为了达成目标，过着一种近乎苦行僧的生活。这也正是很多小镇做题家一直践行的路线，只有努力奔跑，才有可能逆天改命，为自己拼得一个去罗马的人生。相信读到这里会让很多读者产生共鸣。

    # 3.黛西为什么没有选择深情的盖茨比

    不管是看书还是看电影，很多人都会有疑问，为什么黛西不选择盖茨比，又专情，又帅气，而且当时已经很有钱。而自己的丈夫汤姆却滥情出轨，为人也很粗鲁。第一遍看电影的时候这个问题也困惑着我，后来反复思索，有了一定的感悟。

    # (1) 盖茨比并没有真正的跨越自己的阶级，与汤姆黛西始终不是一路人

    有书评对《了不起的盖茨比》一书中作了四层的人物阶级划分。第一层老贵族派，以汤姆黛西为代表，第二层是暴发户新贵派，以盖茨比和居住在西卵的人物为代表，第三层是“我”为代表的典型的职员，第四层是威尔逊，梅朵等底层的社会群体。

    虽然盖茨比努力上进，从第四层跨越到了第二层，却始终无法融入到第一层的老贵族派。比如他始终根植于内心的自卑心态和炫耀财富，比如他多次在汤姆面前的手足无措。在贵族派眼中，是始终看不上盖茨比这些暴发户的。汤姆一直嘲笑红色西服的盖茨比很土，一直质疑他去了牛津是一个可能同名的野鸡大学，而自己出身耶鲁。嫌弃盖茨比称呼别人哥们，是下层人士的口头禅。而黛西呢，一直没有主动的去参加过盖茨比家盛大的宴会，因为在她眼中是看不上他们这些暴发户的举止的，哪怕是应邀去了盖茨比参加一次宴会后，也被盖茨比发现“她并不喜欢今天的宴会”。可以说从羊水中带来的这种阶级鸿沟，注定了他们二人的不可能。

    # (2) 盖茨比钱的来路不正，让黛西最终动摇

    并不能说黛西拜金，而是她从小就习惯了荣华富贵，举手投足之间都是金钱养成的。书中盖茨比有一个论断：黛西的声音里充满了金钱。从小到大的养尊处优让她不可能去过平凡人家的苦日子。而盖茨比的发迹，恰恰来源于买卖私酒(当时美国社会是有禁酒令的)。而在三人摊牌的那晚，汤姆公开了盖茨比的种种不法行为，让黛西动摇了，如果选择了盖茨比，黛西可能随时失去现在的富贵生活。这也是为什么汤姆那么不堪，甚至又粗又笨(第四章中黛西二人吵架时的评价)。黛西也无法离开她的原因。

    # 4.路过心上的句子

    (1) 我们这些薄暮中的年轻职员啊，正在虚度一生中最灿烂的年华、一夜中最美好的时辰。

    (2) “每当你要批评别人，”他告诉我，“要记住，世上不是每个人都有你这么好的条件。”

    (3) 盖茨比信奉的那盏绿灯，是年复一年在我们眼前渐渐消失的极乐未来。我们始终追它不上，但没有关系——明天我们会跑得更快，把手伸得更长……等到某个美好的早晨——于是我们奋力前进，却如同逆水行舟，注定要不停地退回过去。
