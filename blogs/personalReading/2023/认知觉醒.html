    《认知觉醒》是近几年非常火爆的一本个人成长书籍，作者周岭堪称一匹黑马，系统性的介绍了认知觉醒的方方面面，绝对值得作为案头书反复阅读实践。

    # 序言

    无论当代科技如何发达，有一样东西始终无法传递，那就是心智。而人与人之间，最根本的差异就是认知能力上的差异：认知会影响选择，选择决定命运，所以个人成长的本质，就是让大脑的认知越来越清晰。

    # 第一章.大脑--一切问题的起源

    # 第一节 大脑：重新认识你自己

    人的大脑分为三重，起源于爬行动物时代的“本能脑”，起源于哺乳动物时代的“情绪脑”，让人类脱颖而出，成为地球主宰的“理智脑”。

    理智脑非常的发达高级，是人类创造出现代文明与科学的根基。但是理智脑相对于本能脑和情绪脑而言，又非常的弱小，至少有如下的四个原因：

    (1) 形成时间：本能脑有3.6亿年的历史，情绪脑有2亿年的历史，而理智脑只有250万年不到。

    (2) 三重大脑发育成熟的时间不同：本能脑在人出生时就比较完善，情绪脑则要等到青春期才趋于完善，而理智脑则要等到成年早期才基本发育成熟。

    (3) 人体大约有860亿个神经元细胞，而本能脑和情绪脑拥有近八成；

    (4) 本能脑和情绪脑运行速度极快，至少可达11 000 000次/秒，而理智脑的最快运行速度仅为40次/秒，并且理智脑的能耗非常大。

    因此日常的大部分决策，实际上是由本能脑和情绪脑来决定的。然而他们的进化还停留在原始社会时代，他的天性就是目光短浅，即时满足。

    现代社会人的所有成长问题，都可以归结为目光短浅，即时满足的天性上。不过在现代社会，用趋易避难和急于求成来代替更为贴切。

    趋易避难：喜欢做简单和舒服的事情，在核心区周围打转，在舒适区内逃避真正的困难。

    急于求成：凡事希望立刻看到结果，稍有挫折就很快放弃。

    因此个人的成长，其实就是克服天性的过程，学会让理智脑，战胜本能脑和情绪脑。

    # 第二节 焦虑：焦虑的根源

    人类的安全感根源之一，就源于在某一个方面拥有独特的优势：或能力，或财富，或权力，或影响力。在它的驱使下，人们往往就会产生焦虑。

    焦虑的根源也就两个，避难趋易：想不怎么努力就取得成功，急于求成：想同时做很多事情。

    # 第三节 耐心：得耐心者得天下

    提高耐心的两个关键要素：

    复利曲线的存在：

    通过复利曲线这把理性的标尺，可以极大的提高耐心。前期的增长非常缓慢，但是到了拐点之后会飞速增长。

    舒适区边缘是另外一个能力成长的普遍法则：所有人的能力都以“舒适区->拉升区->困难区”的形式分布，要让自己高效的成长，一定要让自己始终处于舒适区边缘。贸然跨到困难区会让自己受挫，而长期处于舒适区就容易停滞不前。

    人的天性却和这个规律相反，急于求成，容易跨越到困难区，导致自己受挫。趋易避难，容易长期停留在舒适区导致止步不前。

    复利曲线和舒适区边缘是一对好朋友，他们组合在一起可以让我们在宏观上看到保持耐心的力量，而这种力量适用于每一个人。

    对于学习成长而言，一定要认识到成长权重的存在。学习之后的思考，思考之后的行动，行动之后的改变更重要。因此学习一定要盯住改变量，如果只在表层投入太多的学习量，只能是事倍功半。比如，读书时不求记住书中的全部知识，只要有一两个观点促使自己发生了切实的改变就足够了。

    一旦认清了上述的规律，在学习的过程中遇到长期的冷寂和挫折时，就会更加容易坚持下去。从这个角度来说，保持耐心其实也是目光长远的体现。

    # 第二章.潜意识--生命留给我们的彩蛋

    # 第一节 模糊：生命是一场消除模糊的大赛

    由于潜意识的处理问题效率，远大于理智脑，因此很多时候的决定，都是潜意识来主导的，但是潜意识没有思维，只有本能，会诱导我们去做更多低能耗的事情。个人成长，就是要不断的去消除模糊，让理智脑更多的参与日常决定。

    消除模糊的核心方法，就是不断的明确核心困难和心得感悟。正如学霸的秘籍在错题本一样。人们常常在模糊区打转，在舒适区兜圈，这都是潜意识在作怪，因为这样做是低能耗的事情。所以一定要明确困难，主动去做高能耗的事情。

    有时候行动力不足的根源，也是由于选择模糊，想做的事情很多，这时候潜意识就会发挥最用，当我们没有足够清晰的指令或者目标时，就很容易选择享乐，放弃那些本该坚持但比较烧脑的选项。消除模糊的一个关键，就是理清任务的优先级，让自己多处于没得选的状态。

    # 第二节 感性：顶级的成长竟然是凭感觉

    我们的理性思考，处理信息的速度只有40次/秒，潜意识是它的275 000倍，二者的能力有着天壤之别。因此潜意识有时候发出的信号，并不能被我们的理性思考所理解。

    在学习中，要善于捕捉自己的潜意识，简单的说，就是当阅读的过程中遇到触发自己的点时，立刻停止下来，想想会什么这里会让自己有感触，能否应用到自己的生活工作当中去。也就是熔断学习法，《认知觉醒》的作者很推崇这种学习模式，每当阅读完一本书，回顾找到最触动自己的点，去反复实践寻求改变。

    熔断学习法，也很符合学习拉伸区的理论：它能帮我们感知真正适合自己并需要的东西，让自己处于学习的“拉伸区”。如果单纯运用理性，我们通常会向优等生看齐，把眼光放在那些最难的题目上，想着如何追赶他们；如果顺从天性，我们就会在最简单的题目上打转。

    # 第三章.元认知--人类的终极能力

    # 第一节 元认知：成长慢，是因为你不会“飞”

    元认知，是最高级别的认知能力，是对自身的思考过程进行认知和理解。

    更通俗的说，元认知就是反思的能力。一旦开启的元认知，就仿佛开启了上帝视角，能够审视你自己的行为是否合理，让理智脑更多的参与日常的决策中去。

    元认知能力总能让你站在高处俯瞰全局，不会让你一头扎进生活的细节，迷失其中。如果你足够细心，还会发现未来视角总是当前行动的指南针，它可以在茫茫的生命中为你导航，让你主动选择去做那些更重要而不是更有趣的事情。

    # 第二节 自控力：我们生而为人就是为了成为思维舵手

    元认知能力就是觉察力和自控力的组合。所以从实用角度讲，元认知能力可以被重新定义为：自我审视、主动控制，防止被潜意识左右的能力。

    实际上，我们天然的被潜意识所左右，总是会出现不受支配的场景。提高元认知的一个关键，就是在选择的节点上，想清楚，让理智脑更多的参与决策。本书提到了一个“元时间”的概念，也就是时间并不是等价的，元时间（也即选择节点的时间）决定了后续时间的质量。在元时间内，一定要更多的让理智脑参与的决策中，这样整个人就会很清晰，不容易受潜意识的控制。

    综上所述，成为思维舵手有3种方法。

    (1) 针对当下的时间，保持觉知，审视第一反应，产生明确的主张；

    (2) 针对全天的日程，保持清醒，时刻明确下一步要做的事情；

    (3) 针对长远的目标，保持思考，想清楚长远意义和内在动机。

    # 第四章.专注力--情绪和智慧的交叉地带

    # 第一节 情绪专注：一招提振你的注意力

    长时间的身心分离模式，会导致我们的生命质量变差，对情绪状态和能力提升产生负面影响。生命是当下的一个个片段组成的，身心合一的片段组成的就是幸福专注的高质量人生。

    提高注意力只有一招，让感受回归行动，不管做什么，都把感受收回来。比如跑步的时候，就享受肢体的摆动和微风的吹拂；吃饭时，就感受每一口饭的香甜。

    # 第二节 学习专注：深度沉浸是进化双刃剑的安全剑柄

    因为沉浸能力的不同，人类最终停留在了不同的层次。

    “正确的方法”通常具有以下四个特征。

    第一：有明确的目标

    以钢琴为例，不要定连续弹奏半个小时这样宽泛的目标，而是形如“连续弹奏三遍不出错，且每分钟120拍”这样的目标。目标越明确，越容易聚焦，精力越集中，越容易精进。

    第二：练习时极度的专注

    大脑在学习的时候有两种模式，一种是“意识”的专注模式，另一种是“潜意识”的发散模式。

    第三：有效的反馈

    想方设法得到及时、有效的指导和反馈是不断精进的重要条件。如果条件有限，反馈也可以通过书籍影像、与他人交流或者自我反思来获取。

    第四：始终让自己处在舒适区边缘

    当人们对当前的活动感到厌倦时，说明应该提高难度；当人们对当前的活动感到焦虑时，说明应该保持这个水平专注练习，如此反复交替就可以让自己进入心流通道，沉浸其中。

    以钢琴为例，作者发现自己的女儿弹琴时，总是犯各种错误。就纠正了练习的方式。当天只练习第一个小节（目标拆解），做到连续弹奏三遍不出错就可以（目标清晰），并且重点练习容易出错的部分（舒适区边缘），练习的过程中，作者也及时的纠正错误的音和手型问题（及时反馈），女儿练习的过程中也更容易专注了，效果也大大的提高。

    # 第五章.学习力--学习不是一味努力

    # 第一节 匹配：舒适区边缘，适用于万物的方法论

    深度学习最核心的要素，就是舒适区边缘的匹配，这样才能更容易让自己容易做到坚持。

    学习也不是一味的打鸡血，拼意志力，只要站在舒适区边缘，一点点往外走，同时间做朋友，肯定会在不经意间发生蜕变。

    # 第二节 深度：深度学习，人生为数不多的好出路

    不同的学习方式之间，留存率和转化率是不同的，如果一直停留在表层的学习，容易只关注表面，投入了大量时间，却没有得到想要的效果。

    不同学习方式之间的留存转化率如下：

    以读书为例：

    从知道到做到是一个巨大的进步，而自己知道到让别人知道是更大的一个跨越。最深层的学习，就是可以通过自己的语言，把学习到的内容给别人也将明白。

    深度学习的三部曲：

    1.获取高质量的知识(减少二手垃圾知识的摄入)

    2.知识的缝接(用自己的话将知识沉淀描述出来)；

    3.输出成果去教授；

    # 第三节 关联：高手的“暗箱”

    新学习到的东西一定要进行关联，否则就难以转换为自己的东西，过一段时间很容易就什么都想不起来了。

    真正学习的过程分为三个步骤：最浅层的获取知识->用自己的话来描述出来，跟已有的知识产生关联->反复的实践，真正转变成为自己的东西。

    在学习的过程中，没必要强求全部理解，因为有的知识本身处于自己的困难区，难以和已有的知识体系建立关联，这种直接丢弃掉也不必可惜。只要能保持持续学习，总能进步。

    另外强迫自己定期写作也是一个非常好的建立关联的手段，能够倒逼自己对已摄入的内容反思沉淀，而普通的阅读和听讲，始终是停留在表层的摄入知识。

    # 第四节 体系：建立个人的认知体系其实很简单

    学习的时候，要保持一个原则，就是重视那些让自己触动的点。

    另外仅仅发生触动还不够，如果没有快速的缝合，触动就只是“伪触动”，过段时间就消失了。深层次的触动包括三个方面：

    (1) 能够用自己的话讲清楚

    (2) 需要的时候能够顺利提取出来，不能提取的都是“伪触动”；

    (3) 能够在日常生活工作中反复实践；

    # 第五节 打卡：莫迷恋打卡，打卡打不出未来

    打卡容易让自己做的事情，产生动机转移。将注意力从做事情本身，转移到为了完成打卡而完成。

    相对于打卡，更好的一个方式是标记。记录下自己的完成情况即可。另外，任务的目标设定，可以采用设下限不设上限的方式，下限可以比较低，这样容易达成，在完成目标的同时，如果有精力可以继续，而且低的下限能够让自己没有太大的压力，更符合舒适区边缘的理论。

    # 第六节 反馈：是时候告诉你什么是真正的学习了

    刻意练习的四原则中，有一个很重要的环节就是及时反馈。及时，持续的正向反馈，可以让我们快速检验成果，取得成就感。让情绪脑和本能脑激发出更大的行动力，让我们享受过程本身。

    对于背记类的学习，自我测试就是最好的反馈，

    另外一个比较好的建立反馈的渠道，就是有作品意识。输出倒闭输入，自己看过学过的东西，可以采用写文章/录视频的方式，产生输出。有了反馈才有持续优化的动力，针对尖锐的反馈，可以考虑是否真的是有价值的建议或者批评，是非常好的帮助自己成长的机会。

    # 第七节 休息：你没成功，可能是因为太刻苦了

    刻意练习的四原则中，第二个就是保持专注，但是专注是非常耗费精力的。通过定时的休息，才可以持续的让自己的精力保持在一个相对高位的水准。

    这里比较推荐的方式是番茄管理法，25分钟专注+5分钟休息的方式。触发休息一定要强制，不能等到非常累的时候才作出休息的决定，就像不能等到口渴才想起来喝水一样。

    # 第六章 行动力--没有行动世界只是个概念

    # 第一节 清晰：一个观念，重构你的行动力

    行动力不足的根本原因，就是目标不够清晰，想做的事情太多。每天早起是一个非常重要的节点，这个时候的大脑处于一个“重启”的状态，在这个时间做好一天的规划是非常重要的。

    建立好的清晰力，有很简单的一招，就是写出来。用一张A4的纸，分为三个区域，顶部按照优先级列下当天要做的事情，下面左侧写下来每个事情大致的时间规划，右侧用来记录实际的完成进度。

    # 第二节 “傻瓜”：这个世界会奖励那些不计得失的“傻瓜”

    只有实际动手去做，才会离目标越来越近。没有任何捷径，也不能奢望快速成功，一个好的习惯建立，通常需要18个月以上，因此切不可急功近利。

    行动的时候，只有突破阈值，才可以看到更高层次的风景。

    # 第三节 行动：“道理都懂，就是不做”怎么破解

    从现在开始，把认知当成技能，知道或想通一个道理时，不要高兴得太早，想想后面还要做大量的练习，这样就不浮躁了。

    现实和理论都告诉我们：懂得百点不如改变一点。真正的成长不在于自己懂得了多少道理，而在于自己改变了多少。

    # 第七章 情绪力--情绪是多角度看问题的智慧

    # 第一节 心智带宽：唯有富足，方能解忧

    稀缺会导致心智带宽被占用过多，从而导致自己作出不理智的决定。因此一定要避免心智带宽被占用过多，“少即时多”。当自己的脑子一团乱麻时，可以采用把所有的想法写出来的方式，辅助自己理清思路。

    # 第二节 单一视角：你的坏情绪，源于视角单一

    用相机机位的角度，可以更好的阐述视角。例如针对一个画面，即使是使用同一个相机，不同的拍摄角度，拍出来的效果也会有非常大的差异。更何况不同的相机配置也不同。不同的人看问题的角度，就可以类比为不同配置的相机。

    针对同一个事情，时常换用不同的视角，就可以给自己带来不同的感受。例如领导安排了很困难的工作，换个角度想，是自己非常好的成长机会；与一个大领导1v1，换个角度可以认为是一个与远高于自己水平的人深入对话的机会。常常有这种换视角的想法，就能将很多坏情绪给化解掉。

    与不同水平的人沟通时，也可以利用换机位的方式来处理。人的认知层次是不一样的，如果自己的认知高于别人，这时候需要做的是降低自己的层次到他的视角，才能使沟通更好的进行下去，而不是一味的指责谩骂。好的交流都是客观的、不带主观猜测的交流，这样才会让双方都摆脱“战斗模式”。

    # 第三节 游戏心态：幸福的人，总是在做另外一件事

    游戏心态其实就是积极心理学，作者在这里举了一个例子，1500米跑步，将他拆分为一个又一个的10米。问题的难度瞬间就化解了。非常符合舒适区边缘的理论。

    推而广之，游戏心态可以简单理解为：跳出事情本身，这句“咒语”便是：我并不是在做这件事，我只是在做另外一件事。在心理学上，就叫做“动机转移”。

    # 第八章 早冥读写跑--人生五件套

    # 第一节 早起：无闹钟、不参团、不打卡，我是如何坚持早起的

    睡眠节点理论，人在睡眠3，4.5，6，7.5小时这几个时间点，是比较好的临界点。有时候虽然睡觉的时间很久，但起来却依然感觉比较困，原因可能就是没有在睡眠临界点醒来。

    # 第二节 冥想：终有一天，你要解锁这条隐藏赛道

    不难想象，成绩好的人的真正优势在于，他们能够长时间让“7个小球”同时关注一件事情，以保证高质高效的学习，而在成绩不好的人脑中，很可能一个球在播放背景音乐，一个球在想晚上吃什么，一个球在担心即将到来的考试……真正用于学习的小球或许只有三四个。

    冥想的作用，就是把七个小球拉到同一个赛道上。

    # 第三节 阅读：如何让自己真正爱上阅读

    阅读是一个非常好的与行业大师对话的机会，善于阅读，能够帮助我们快速进阶一个领域。

    第一步：善于选书。一定要结合自己当前的需求，去选择行业最顶尖的书籍。

    第二步：阅读要看中转换率。一味的追求数量，只会徒增焦虑。看完书之后只是浅层的摄入，要重视后续的思考，实践，输出三个环节，才是真正的做到了深度学习。

    第三步：读写不分家，阅读后，尝试用自己的话再写出来，让别人也能看懂，写作的过程也能调动自己深度思考的能力。

    # 第四节 写作：谢谢你，费曼先生

    写作的过程要善于类比，用最简单的语言，让没有什么基础的人也能看懂，才是最终极的写作技巧。

    # 第五节 运动：灵魂想要走得远，身体必须在路上

    运动之后，进行高密度的用脑活动，可以物理上让人变聪明。“健身其实也是健脑”，只不过要重视运动后的用脑。推荐的方式是在早上运动，然后在上午完成比较困难的工作。

    # 结语.一流的生活不是富有，而是觉知

    实践“每日反思”，是一个自我觉知的新出口。它相当于一个深度日志记录，每天只需花一点点时间，对当天最触动自己的事情或感悟进行复盘，就可以保持对生活更深的觉知。

    我的方法很简单，就是留意每天生活中最触动自己的点。不管这个点是令人欣喜的感悟，还是令人难受的困惑，只要它在心头燃起火花，就把它摘取下来，记录到文档里复盘。而复盘的方式也极为简单，通常只需3点：

    ①描述经过——以便日后回顾时能想起当时的场景；

    ②分析原因——多问几个为什么，直到有深度的启发；

    ③改进措施——尽可能提炼出一个认知点或行动点。

    # 全书对自己最有触动的点

    # 认知大脑的结构

    三重大脑结构的存在，可以解释很多问题。认知觉醒的过程，就是更多的让理智脑，战胜情绪脑和本能脑，从而改变“趋易避难，急于求成”的天性弱点。

    # 学习方式的改变

    深度学习的四个关键点：明确可量化的目标，高度专注，舒适区边缘练习，及时反馈。

    重视学习的转化率，以读书为例，表面上读完一本书，只是最浅层的知识获取，后续的思考->实践->教学，才是完成深度知识转换的关键环节。后续读书不能只追求数量的多少，而是要重视该变量，一本书哪怕只有一个知识点让自己产生了触动，并形成了改变，就是有价值的，其他的哪怕再有道理，距离自己当前的需求太远或者处于困难区难以吸收，那么丢弃了也不用觉得可惜。

    # 清晰力

    一招建立清晰力，每天早上写下自己当前的待办事情，预计完成时间，当天再整理下实际的完成情况。

    行动力不强的根本原因，其实就是不够清晰。让自己处于没得选的过程，就能更加高效的利用时间。

    # 每日复盘反思

    全书最重要的技能。通过反思，可以提高自己对细节的感知能力，并且及时捕获大脑一闪而过的想法，间接的相当于节省了自己的时间。

    每日复盘也符合几个基本盘：

    (1) 触动学习法，捕获最打动自己的点，从而让自己更高效的学习

    (2) 运用了元认知，可以从另外一个视角，看待自己的每一天

    (3) 遵循了“刻意练习”的原则。刻意练习的要义之一就是带着清晰的目标去学习。生活也是一样的，如果只是随波逐流，不动脑子地度过一天又一天，我们顶多是增长年龄，但如果能带着要领去生活，我们就会成长飞快。
